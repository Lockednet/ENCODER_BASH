# ENCODER

## INSTALAR ENCODER

```
sudo apt-get update && sudo apt-get install build-essential && wget https://github.com/Lockednet/ENCODER_BASH/raw/main/encoder.sh && chmod 777 encoder.sh && ./encoder.sh
```

## PARA ACESSAR OUTRAS VEZES 

```
./encoder.sh
```
